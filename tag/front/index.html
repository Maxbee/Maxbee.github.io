css 让浏览器的滚动条一直显示

2021-06-15 #前端
chrome浏览器地上的滚动条只会在你滚动的时候显示滚动条，如果想让滚动条默认显示可以用如下代码 &lt;div class=&quot;test&quot;&gt; 滚动的区域 &lt;/div&gt; .tester::-webkit-scrollbar { /* 隐藏默认的滚动条 */ -webkit-appearance: none; } .tester::-webkit-scrollbar:vertical { /* 设置垂直滚动条宽度 */ width: 2px; } .tester::-webkit-scrollbar:horizontal{ /* 设置水平滚动条厚度 */ height: 2px; } .tester::-webkit-scrollbar-thumb { /* 滚动条的其他样式定制，注意，这个一定也要定制，否则就是一个透明的滚动条 */ border-radius: 8px; border: 2px solid rgba(255,255,255,.4); background-color: rgba(0, 0, 0, .5); }
