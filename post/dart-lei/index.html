类的继承

1 、子类使用extends关键字来继承父类
2、子类会继承父类里面可见的属性和方法，但是不会继承构造函数，需要重写
3、子类能复写父类的方法 getter 和setter

Dart私有变量在变量名前加下划线 _ , 私有变量需要把类抽离到单独的文件才不能访问，否则调用还是可以访问私有变量。

class Person{
    String _name;
    //构造函数
    Person(this.name)
    //命名构造函数
    Person.getName(){
        print("$_name")
    }
}

命名可选参数大括号：

大括号之内的是选填参数，可以不填，不填为空。例如：

void main(){
  test(name:"xiegm");
}
 
void test({String name,String age}){
  print('名字是$name，今年$age岁了');
}

输出的结果是：名字是xiegm，今年null岁了

位置可选参数：

void main(){
  test("1");
}
 
void test(String day,[int age,String name,]){
  print('今天是星期$day,我是$name，我今年$age岁了');
}

输出结果是：今天是星期1,我是null，我今年null岁了

第一个参数day是必填的，如果要填第三个参数，必须要填第二个参数。如果填入的数据为(String,String)语法检测会不通过

默认值：

位置选填参数：

  void test(String day,[int age=10,String name,])
